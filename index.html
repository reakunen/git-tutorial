Cloning

  • make a repository first on github
  • use git clone (https url or ssh) on the terminal:
    looks like: git clone https://github.com/reakunen/test-aws.git

Forking

  • Fork someone else's repository
  • This means you have no access to their repo, so you make a "clone" of theirs
  • This can be done through the web page
  • OR log into github, gh auth login --web
  • Git cloning: gh repo fork https://github.com/learn-devops-fast/spock-lizard-docker.git --clone
